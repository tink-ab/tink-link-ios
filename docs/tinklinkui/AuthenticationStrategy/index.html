Enumeration Authentication​Strategy 

public enum AuthenticationStrategy

AuthenticationStrategy represents the authentication strategy to use when authenticating the user towards Tink Link. An access token would be used directly to initialize the user session. An authorization code would be exchanged for an access token and then initialize the user session.
